Format ini mengurangi penggunaan token hingga 30-60% dibandingkan JSON sambil mempertahankan keterbacaan dan struktur. Format TOON menggunakan format tabular untuk array dan sintaks TOON minimal untuk objek, membuat konversi JSON ke TOON sangat efisien.

TOON biasanya menghemat 30-60% token LLM dibandingkan dengan JSON. Untuk dataset besar atau respons API, ini dapat menghasilkan penghematan biaya yang signifikan saat menggunakan API seperti GPT-4 atau Claude. Penghematan yang tepat tergantung pada struktur data Anda.
