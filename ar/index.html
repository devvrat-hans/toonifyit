الأسئلة الشائعة

TOON (Token-Oriented Object Notation) هي صيغة تسلسل بيانات مصممة خصيصاً لموجهات LLM. تقلل استخدام الرموز بنسبة 30-60% مقارنة بـ JSON مع الحفاظ على سهولة القراءة والبنية. تستخدم صيغة TOON تنسيقاً جدولياً للمصفوفات وبنية بناء جملة TOON الحد الأدنى للكائنات، مما يجعل تحويل JSON إلى TOON فعالاً جداً.

عادة ما يوفر TOON 30-60% على رموز LLM مقارنة بـ JSON. بالنسبة لمجموعات البيانات الكبيرة أو استجابات API، يمكن أن يترجم هذا إلى توفير كبير في التكاليف عند استخدام واجهات برمجية مثل GPT-4 أو Claude. يعتمد التوفير الدقيق على هيكل بيانات.

نعم، يعمل TOON مع جميع LLMs الرئيسية بما فيها GPT-4 و Claude و Gemini و LLaMA. منذ TOON هي صيغة نصية، يمكن لأي LLM قراءتها وفهمها. يمكن لمعظم LLMs الحديثة تحليل صيغة TOON بتعليمات بسيطة في الموجه.

بالتأكيد. تتم جميع التحويلات محلياً في متصفحك باستخدام JavaScript. لا تترك بيانات جهازك أبداً أو يتم إرسالها إلى أي خادم. يعمل المحول بشكل كامل دون اتصال بالإنترنت بمجرد تحميل الصفحة.

JSON مقابل TOON: يستخدم TOON صيغة أكثر إحكاماً مصممة لكفاءة الرموز. يمثل تنسيق تسلسل TOON المصفوفات كجداول مع رؤوس، ويستخدم علامات ترقيم بسيطة جداً، ويلغي الأحرف الزائدة. JSON أكثر تفصيلاً مع أقواس وعلامات اقتباس وفواصل. عند تحويل JSON إلى TOON لموجهات LLM، تقلل كفاءة TOON التكاليف بشكل كبير. اقرأ مقارنتنا التفصيلية بين TOON و JSON.

تحويل JSON إلى TOON بسيط: 1) الصق بيانات JSON الخاصة بك في محولنا أعلاه، 2) انقر فوق "تحويل إلى TOON"، 3) انسخ إخراج TOON. للتحويل البرمجي، يمكنك استخدام مكتبات JavaScript أو Python الخاصة بنا. اطلع على دليل تحويل JSON إلى TOON الكامل مع أمثلة وأجزاء من الكود.

نعم! TOON قابلة للعكس بالكامل - يمكنك تحويل TOON مرة أخرى إلى JSON دون فقدان أي بيانات. يدعم محولنا ثنائي الاتجاه تحويل JSON إلى TOON و TOON إلى JSON. ما عليك سوى لصق بيانات TOON الخاصة بك واختيار "تحويل إلى JSON" للحصول على هيكل JSON الأصلي.

يعمل TOON بشكل أفضل مع البيانات الجدولية الموحدة مثل نتائج استعلامات قاعدة البيانات واستجابات API مع قوائم الكائنات وبيانات التحليلات ومجموعات بيانات على غرار CSV وكتالوجات المنتجات في التجارة الإلكترونية. تحصل البيانات مع الحقول المتكررة على أعلى توفير رموز (40-60%). قد تحصل مجموعات البيانات الصغيرة أو الهياكل المتداخلة بعمق على تحسينات أصغر.

نعم، يدعم TOON الكائنات والمصفوفات المتداخلة. تستخدم الكائنات المتداخلة بنية قائمة على المسافة البادئة (مثل YAML)، بينما تحافظ المصفوفات المتداخلة على بنيتها. ومع ذلك، فإن كفاءة الرموز في TOON أعلى مع البيانات المسطحة والموحدة. تعمل الهياكل المتداخلة المعقدة ولكن قد لا ترى انخفاضاً كبيراً في الرموز.

نعم، محول TOON والصيغة TOON نفسها مجانية تماماً للاستخدام. لا توجد حدود على التحويلات أو تقييدات حجم الملف أو ميزات متميزة. مواصفات صيغة TOON مفتوحة ومتاحة لأي شخص لتنفيذها في تطبيقاته.

يعمل TOON مع جميع موفري LLM الرئيسيين بما في ذلك OpenAI (GPT-4، GPT-3.5)، Anthropic (Claude)، Google (Gemini)، Meta (LLaMA)، وآخرين. منذ TOON هي صيغة نصية، يمكن لأي LLM فهم TOON بتعليمات الموجه المناسبة الذي يشرح هيكل الصيغة.

يقلل TOON الرموز من خلال ثلاث عمليات تحسين رئيسية: 1) التصريح بأسماء الحقول مرة واحدة في رؤوس المصفوفة بدلاً من تكرارها لكل كائن، 2) استخدام علامات ترقيم بسيطة جداً والقضاء على الأقواس والأقواس غير الضرورية، 3) استخدام بنية قائمة على المسافة البادئة بدلاً من الفواصل الصريحة. يتحد هذا لتحقيق تقليل بنسبة 30-60% من الرموز مقارنة بـ JSON.

بالتأكيد! يعمل TOON بشكل مثالي مع ChatGPT API (GPT-4، GPT-3.5-turbo). قم ببساطة بتحويل بيانات JSON الخاصة بك إلى TOON قبل إدراجها في الموجه. أضف شرحاً موجزاً لصيغة TOON في رسالتك النظام، وسيفهم النموذج ويعالج البيانات. يمكن أن يقلل هذا بشكل كبير من تكاليف API للموجهات الثقيلة بالبيانات.

عادة ما تستخدم ملفات TOON امتداد .toon (على سبيل المثال، data.toon). ومع ذلك، منذ TOON هي صيغة نصية، يمكنك أيضاً حفظها كملفات .txt. عند التنزيل من محولنا، الامتداد الافتراضي هو .toon لتحديد نوع الصيغة بوضوح.

التكامل بسيط: 1) قم بتثبيت مكتبة TOON للغتك (npm install @toon-format/toon للـ JavaScript، pip install toon-format للـ Python)، 2) استيراد وظائف encode/decode، 3) تحويل بيانات إلى TOON قبل الإرسال إلى LLM، 4) يمكن تحويل الاستجابات مرة أخرى إلى JSON. راجع التوثيق الخاصة بنا للحصول على أمثلة على الكود وسند API.

نعم، يعمل TOON مع استجابات بث LLM. تستخدم TOON لبيانات الإدخال في موجهاتك لتقليل رموز الإدخال. يمكن أن تكون استجابة LLM (سواء كانت بث أم لا) بأي تنسيق - TOON مصممة بشكل أساسي لتحسين رموز الإدخال الخاصة بك. إذا كنت تحتاج إلى إخراج TOON من LLM، أضف تعليمات التنسيق في موجهك.

يمكن لمحولنا عبر الإنترنت التعامل مع الملفات حتى 10MB، وهو مناسب لمعظم حالات الاستخدام. بالنسبة لمجموعات البيانات الأكبر، نوصي باستخدام أداة سطر الأوامر أو دمج مكتبة TOON مباشرة في تطبيقك. المكتبة نفسها لا توجد حدود لحجم الملف ويمكنها معالجة مجموعات البيانات الكبيرة جداً بكفاءة.
