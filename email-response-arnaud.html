Why TOON Achieves Better Accuracy

The key is TOON's design, which reduces the token count by 30-60%, allowing models to focus on meaningful data rather than repetitive syntax.

Official benchmarks across models like GPT-5 Nano and Claude Haiku show TOON achieving an accuracy of around 68.7-73.9%, compared to JSON's 65.4-69.7%, all while using significantly fewer tokens.

A detailed study tested 201 real-world questions and found that TOON used 39.5% fewer tokens, consistently achieving better accuracy.

How TOON Works

TOON works best with uniform, tabular data by declaring the schema once and listing data compactly, unlike JSON that repeats key names over and over. This clarity helps LLMs parse faster and more accurately.

Example: For tabular data with 100 rows, JSON repeats field names 100 times, while TOON declares them once in the header. This structural efficiency leads to both fewer tokens and better comprehension by LLMs.
