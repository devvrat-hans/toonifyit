2

3

Sử dụng nút "Sao Chép" để sao chép đầu ra TOON vào bộ nhớ tạm của bạn, hoặc nhấp vào "Tải Xuống" để lưu dữ liệu JSON được chuyển đổi sang TOON dưới dạng tệp.

TOON (Token-Oriented Object Notation) là một định dạng tuần tự dữ liệu được thiết kế đặc biệt cho các prompt LLM. Nó giảm sử dụng token 30-60% so với JSON trong khi duy trì khả năng đọc và cấu trúc. Định dạng TOON sử dụng định dạng bảng cho mảng và cú pháp TOON tối thiểu cho các đối tượng, làm cho chuyển đổi JSON sang TOON cực kỳ hiệu quả.

TOON thường tiết kiệm 30-60% token LLM so với JSON. Đối với các bộ dữ liệu lớn hoặc phản hồi API, điều này có thể dịch thành tiết kiệm chi phí đáng kể khi sử dụng các API như GPT-4 hoặc Claude. Tiết kiệm chính xác phụ thuộc vào cấu trúc dữ liệu của bạn.

Có, TOON hoạt động với tất cả các LLM chính bao gồm GPT-4, Claude, Gemini và LLaMA. Vì TOON là định dạng văn bản, bất kỳ LLM nào cũng có thể đọc và hiểu nó. Hầu hết các LLM hiện đại có thể phân tích định dạng TOON với các hướng dẫn đơn giản trong prompt của bạn.
