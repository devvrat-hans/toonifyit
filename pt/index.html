Perguntas Frequentes

TOON (Token-Oriented Object Notation) é um formato de serialização de dados projetado especificamente para prompts de LLM. Reduz o uso de tokens em 30-60% em comparação com JSON enquanto mantém a legibilidade e estrutura. O formato TOON utiliza um formato tabular para arrays e sintaxe TOON mínima para objetos, tornando a conversão de JSON para TOON altamente eficiente.

TOON tipicamente economiza 30-60% em tokens LLM em comparação com JSON. Para conjuntos de dados grandes ou respostas de API, isso pode se traduzir em economias de custo significativas ao usar APIs como GPT-4 ou Claude. A economia exata depende da estrutura dos seus dados.

Sim, TOON funciona com todos os LLMs principais incluindo GPT-4, Claude, Gemini e LLaMA. Como TOON é um formato de texto, qualquer LLM pode lê-lo e entendê-lo. A maioria dos LLMs modernos pode analisar o formato TOON com instruções simples no seu prompt.

Absolutamente. Todas as conversões ocorrem localmente no seu navegador usando JavaScript. Seus dados nunca saem do seu dispositivo nem são enviados para nenhum servidor. O conversor funciona completamente offline uma vez que a página está carregada.

JSON vs TOON: TOON utiliza uma sintaxe TOON mais compacta projetada para eficiência de tokens. O formato de serialização TOON representa arrays como tabelas com cabeçalhos, usa pontuação mínima e elimina caracteres redundantes. JSON é mais verboso com colchetes, aspas e vírgulas. Quando você converte JSON para TOON para prompts LLM, a eficiência do TOON reduz significativamente os custos.
