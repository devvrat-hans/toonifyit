Vanliga frågor

TOON (Token-Oriented Object Notation) är ett dataserialiseringsformat utformat specifikt för LLM-prompts. Det minskar tokenanvändningen med 30-60% jämfört med JSON medan läsbarheten och strukturen behålls. TOON-formatet använder ett tabellarformat för matriser och minimal TOON-syntax för objekt, vilket gör JSON till TOON-konvertering mycket effektiv.

TOON sparar vanligtvis 30-60% på LLM-tokens jämfört med JSON. För stora datamängder eller API-svar kan detta översättas till betydande kostnadsbesparingar när du använder API:er som GPT-4 eller Claude. De exakta besparingarna beror på din datastruktur.

Ja, TOON fungerar med alla större LLM:er inklusive GPT-4, Claude, Gemini och LLaMA. Eftersom TOON är ett textformat kan alla LLM läsa och förstå det. De flesta moderna LLM:er kan tolka TOON-format med enkla instruktioner i din prompt.

Absolut. All konvertering sker lokalt i din webbläsare med JavaScript. Dina data lämnar aldrig din enhet eller skickas till någon server. Konverteraren fungerar helt offline när sidan har laddats.

JSON vs TOON: TOON använder en mer kompakt TOON-syntax utformad för tokeneffektivitet. TOON-serialiseringsformatet representerar matriser som tabeller med rubriker, använder minimal interpunktion och eliminerar redundanta tecken. JSON är mer verbose med hakparenteser, citattecken och kommatecken. När du konverterar JSON till TOON för LLM-prompts minskar TOONs effektivitet kostnaderna avsevärt. TOON fungerar bäst med enhetliga tabellariska data som databasfrågresultat, API-svar med listor med objekt, analysdata, CSV-stiliserade datamängder och e-handelsproduktkataloger. Data med upprepade fält över flera objekt ser de högsta tokenbesparingarna (40-60%). Små datamängder eller djupt kapslade strukturer kan se mindre förbättringar.

Ja, TOON stöder kapslade objekt och matriser. Kapslade objekt använder indenteringsbaserad struktur (som YAML), medan kapslade matriser behåller sin struktur. TOONs tokeneffektivitet är högst med platta, enhetliga data. Komplexa kapslade strukturer fungerar fortfarande men kan inte se lika dramatisk tokenminskning.

Ja, både TOON-konverteraren och TOON-formatet är helt gratis att använda. Det finns ingen gräns för konverteringar, filstorleksbegränsningar eller premiumfunktioner. TOON-formatspecifikationen är öppen och tillgänglig för vem som helst att implementera i sina applikationer.

TOON fungerar med alla större LLM-leverantörer inklusive OpenAI (GPT-4, GPT-3.5), Anthropic (Claude), Google (Gemini), Meta (LLaMA) och andra. Eftersom TOON är ett textbaserat format kan alla LLM som kan bearbeta text förstå TOON med lämpliga instruktioner som förklarar formatstrukturen.

TOON minskar tokens genom tre nyckeloptimieringar: 1) Deklarera fältnamn en gång i matrisbeteckningar istället för att upprepa dem för varje objekt, 2) Använd minimal interpunktion och eliminera onödiga parenteser/klamrar, 3) Använd indenteringsbaserad struktur istället för explicita avgränsare. Dessa kombineras för att uppnå 30-60% tokenminskning jämfört med JSON.

Absolut! TOON fungerar perfekt med ChatGPT API (GPT-4, GPT-3.5-turbo). Konvertera helt enkelt dina JSON-data till TOON innan du inkluderar det i din prompt.
